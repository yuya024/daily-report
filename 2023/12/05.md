# わかったこと

- XSS は script のサニタイズを行うことによって防ぐことができる。手口として script を実行させて偽サイトに誘導して個人情報などを入力させて盗むやり方がある
- CSRF はトークンなど秘密情報を発行することで防ぐことができる。手口としてログインした状態で、偽サイトから意図しないリクエストが送信されることがある
- メールヘッダインジェクションは直接ヘッダに入力値を含めることで起きてしまう
